# 🛡️ CS50 Introduction to Cybersecurity - Study Notes & Insights

Welcome to my personal study repository! This repository contains my comprehensive notes, concept breakdowns, and practical insights compiled while completing the **CS50's Introduction to Cybersecurity** course.

The goal of this repository is to document my learning journey and solidify my understanding of core security principles.

---

## 🗺️ Course Syllabus & Repository Map

The notes are organized by modules following the official course layout:

* 📁 **01-Securing-Accounts:** Deep dive into authentication protocols, password security, hashing, salting, and two-factor authentication (2FA).
* 📁 **02-Securing-Data:** Exploring cryptography, symmetric/asymmetric encryption, and how data is protected both at rest and in transit.
* 📁 **03-Securing-Systems:** Network security configurations, understanding firewalls, endpoint protection, and basic defensive tools.
* 📁 **04-Securing-Software:** Common software vulnerabilities, input validation techniques, SQL injections, and buffer overflow concepts.
* 📁 **05-Preserving-Privacy:** Digital footprints, anonymity online, and the ethical/legal frameworks surrounding modern data privacy.
